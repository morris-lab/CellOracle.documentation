Google Colaboratoly¶

Let’s try CellOracle tutorials in Google Colaboratory environment. You can easily run celloracle using built-in dataset.

  • scRNA-seq data: hematopoiesis data published by Paul et.al (2015).

  • base GRN: mouse sci-ATAC-atlas dataset

Setup CellOracle in Google Colab¶

celloracle uses several python libraries and R library. Please run the notebook in Google Colab. All dependent software will be installed.
